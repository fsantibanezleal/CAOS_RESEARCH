# EXP-003 - A general seed-family constructor, derived and verified by us

- **Question.** Does the structure established in EXP-002 extend to a CONSTRUCTOR: seed polynomial
  p(w) (p(0) = 0) plus constants (k, m) -> a Keller map F_{p,k,m} on C^3? For which (p, k, m)?
  Does it produce genuinely NEW counterexamples (higher fiber degree) with explicit rational
  collisions?
- **Motivation.** Felipe's directive: focus on generalization. EXP-002 showed F is generated by
  (b1, c1) over the invariant plane with a reduced 2-variable Keller identity. The candidate
  constructor (our reverse-derivation, to be tested):
  - h(w) = p(w)/w, Phi(w) = integral_0^w p;
  - c1 = h(v) - t; w-coordinate w = u c1 / k (u = 1 + v);
  - b1 = k u h(u c1 / k) + m;
  - a1 = u b1 / k - k^2 Phi(u c1 / k) / c1^2 (polynomial since Phi starts at w^2);
  - F_{p,k,m} = (a1/x^2, b1/x, x c1) under v = xy, t = x^2 z.
  It reproduces F at p = 2w - 3w^2, k = 2, m = 2.
- **Falsifiable predictions.**
  1. For seed degree d = 2 with general (p1, p2, k, m), imposing (a) polynomiality of all three
     components in (x, y, z) and (b) constant nonzero det J yields a nonempty constraint set
     containing (2, -3, 2, 2); the constraints will pin the relations among (p, k, m) (we expect
     conditions equivalent to p(1) = -c and integral_0^1 p = 0 to appear, but the run decides).
  2. The same holds for d = 3, and the instance p = w - 2w^3 admits valid (k, m) giving a Keller
     map G with constant det and generic fiber degree 4.
  3. The reconstruction inverse (from a root w of the fiber polynomial back to (x, y, z)) is
     exact, and engineered targets give EXPLICIT RATIONAL COLLISIONS for G: distinct rational
     points with identical images, i.e. a new counterexample certificate not in the announcement.
  4. A degree-4 seed instance can be built the same way (fiber degree 5), fully verified.
- **Method.** sympy over QQ. Undetermined-coefficient solves for the constraint sets; exact det
  computation; fiber degree via the w-polynomial Phi(w) - (w Vt - Tt)/k^2 at random rational
  targets plus exact reconstruction and re-substitution; collision engineering by prescribing two
  rational w-roots and solving the (linear) system for the target invariants.
- **Success criterion.** Predictions 1-3 verified exactly (4 if runtime allows).
- **Failure criterion.** The constructor fails Keller or polynomiality for ALL (k, m) at some
  seed satisfying the discovered conditions, or reconstruction mismatches: then our reconstruction
  is wrong and the verdict documents where.
