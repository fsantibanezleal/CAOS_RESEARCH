# EXP-091 - Verdict: JC(2) sits next to Rentschler and Abhyankar-Moh (via LNDs)

**Status: MAPPED [MV bracket; D reductions]. Known connections, pinned for our
program. No novelty claimed for the connections.**

- [MV] J(P,Q) = {P,Q} identically; planar Keller pairs are exactly polynomial
  canonical pairs ({P,Q}=1). JC(2) <=> every canonical pair generates C[x,y].
- [D, known] D = {P,.} is a derivation with D(Q)=1. IF D is locally nilpotent, then
  by Rentschler's classification of LNDs of C[x,y] (every LND ~ p(x) d/dy up to a
  polynomial automorphism), D(Q)=1 forces D ~ d/dy, so P is a coordinate and the
  pair is an automorphism.
- [D, known] D is an LND iff its Hamiltonian flow is a polynomial G_a-action iff the
  generic fibre {P=c} is a line A^1. So, on this route, JC(2) reduces to: does
  {P,Q}=1 force the P-fibres to be lines? That is the Abhyankar-Moh-Suzuki
  embedded-line problem (an embedded line in A^2 is rectifiable - a THEOREM; the
  open part is whether a Keller P has line fibres).

## Value for our program

adjacent to two classification theorems (Rentschler; Abhyankar-Moh-Suzuki) and
