# Expanded research lines for JC(2) (lens-driven, 2026-07-24)

Beyond the exclusion spine and the incidence/recognition work (S7-S9, EXP-086-090),
new points of view to develop. Each tagged by lens (methodology/10) and with a
first concrete step. None is a claim; each is a declared direction.

## L1. Poisson / symplectic reformulation [reformulation]
J(P,Q)=1 IS {P,Q}=1 for the standard symplectic form dx^dy. So a Keller pair =
a polynomial "canonical coordinate pair", and JC(2) <=> every polynomial pair with
{P,Q}=1 generates C[x,y] (is a global coordinate system / polynomial
symplectomorphism). The 3D counterexample is not symplectic (odd dim); the planar
case is the genuinely symplectic one. First step (EXP-091): recast the (72,108)
reduced system as a symplectic-bracket condition and test whether the Poisson
structure adds constraints the covector sweep misses (e.g. {P,Q}=1 forces P,Q to
be a Darboux pair locally; obstruction to globalizing).

## L2. Quantization / Weyl-algebra bridge [reformulation]
Quantize {,} to the commutator of the first Weyl algebra A_1. Belov-Kanel-
Kontsevich: JC(2n) is stably equivalent to Dixmier(n) (endomorphisms of A_n are
automorphisms). JC(2) itself is not literally Dixmier, but our annihilation lemma
J(m,P^k)=-kL(P^{k-1}m) is a classical-limit connection (EXP-079). First step:
compute the "quantum obstruction" - does the (72,108) covector obstruction have an
A_1-module interpretation (the flat-connection lead made noncommutative)?

## L3. Jelonek non-properness curve [reformulation + our EXP-014]
A planar Keller counterexample is non-proper; its Jelonek set is a curve that is
C-uniruled (Jelonek). Cross this with Newton-polygon data: the asymptotic set is
the image of the boundary at infinity, which our polygon machinery already sees.
First step (EXP-092): for the incidence-type maps we understand, compute the
Jelonek curve = discriminant, and check whether the uniruledness + our
weight-class theorems are jointly contradictory for a hypothetical planar pair.

## L4. Recognition obstruction, generalized [recognition + invariant]
EXP-089/090 killed the incidence class via "X_2 != A^2 (units/reducible)". The
general planar counterexample would still need a source recognized as A^2. Miyanishi-
Sugie classifies A^2. First step: secure Miyanishi-Sugie; state the general
recognition obstruction a planar Keller counterexample's source must evade, and
whether ANY known exotic-A^2-like surface admits an etale degree-d map to A^2.

## L5. Sheet-number / degree of the cover [dimension-ladder + at-infinity]
A planar Keller counterexample is a d-sheeted cover, d>1. Newton-polygon +
one-place-at-infinity (Abhyankar-Moh) constrain d. First step: bound the sheet
number of a hypothetical (72,108) pair from the reduced polygon; if d is forced =1,
that IS invertibility.

## L6. char-p fast filter [reformulation]
Reduce mod p and use point-counts / Frobenius as a cheap necessary filter on
candidate reduced systems (our 3D char-p certificates, ported to 2D). First step:
a mod-p point-count sanity on the (72,108) reduced system vs an invertible control.
